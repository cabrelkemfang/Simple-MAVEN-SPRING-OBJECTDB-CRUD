This is a simple to intermediate sample of Create,Retrieve,Update and Delete (C.R.U.D) operations using JAVA programming language where it is powered by Maven Project ,Spring Framework and ObjectDB as back-end. This project also is a web application which is using a Web Framework Bootstrap, which composed of Updated web technologies such as HTML 5,CSS 3,JavaScript/JQuery.
